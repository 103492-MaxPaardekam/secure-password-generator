About & Security

Encryption

Keysmith uses industry-standard encryption:

  • PBKDF2 with 600,000 iterations for key derivation
  • AES-256-GCM authenticated encryption
  • Web Crypto API for secure randomness

Local-First Storage

Your data never leaves your device:

  • All data stored in IndexedDB
  • No network requests — ever
  • No analytics or tracking
  • Works offline as a PWA

Limitations

  • Clipboard: Cannot guarantee clearing on all browsers
  • Memory: Decrypted data in memory while unlocked
  • Backups: You must backup your vault
  • Lost Password: Cannot be recovered

Recommendations

  • Use a strong, memorable master password
  • Export backups regularly
  • Store backups in multiple locations
  • Run security audits periodically
